# **App Name**: EduCloud

## Core Features:

- User Authentication: Secure user login and signup using Firebase Authentication.
- Course Management: Admin dashboard to create, update, and delete courses.
- Content Access: Student interface to view course content, lessons, and modules.
- Quiz System: Students can attempt quizzes and track their scores. Admin can add, update and remove quizzes
- Progress Tracking: Track student progress on course content and quiz scores, stored in Firestore.
- Admin Authorization: Secure admin roles and permissions to protect management functionality, enforced by tool use in server actions.
- Resource Downloads: Allow students to download lecture materials and other learning aids

## Style Guidelines:

- Primary color: Indigo (#4F46E5) for a modern, trustworthy feel.
- Background color: Light gray (#F9FAFB), creating a clean and professional space.
- Accent color: Green (#34D399), used for positive feedback, success messages, and highlights.
- Headline font: 'Space Grotesk' sans-serif for headlines; modern and clean. Body font: 'Inter' sans-serif for readability; should also be used when the situation requires small blocks of text.
- Lucide Icons, simple and consistent for clear navigation.
- Responsive layout for seamless access on all devices.
- Subtle fade-in effects for a smooth user experience.
